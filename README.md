- [OWASP](https://owasp.org) - Open-source foundation improving software security.
- [PentesterLab](https://pentesterlab.com) - Hands-on web penetration testing exercises.
- [VulnHub](https://vulnhub.com) - Free vulnerable VMs to practice on locally.
- [TCM Security](https://academy.tcm-sec.com) - Affordable, high-quality ethical hacking courses.
- [Cybrary](https://cybrary.it) - Free and paid IT and cybersecurity courses.

## 🔍 Reconnaissance

- [Shodan](https://shodan.io) - Search engine for internet-connected devices.
- [Censys](https://censys.io) - Internet-wide scanning data and certificates.
- [theHarvester](https://github.com/laramies/theHarvester) - OSINT tool for gathering emails, subdomains, hosts, etc.
- [Maltego](https://maltego.com) - Visual link analysis and OSINT tool.
- [Recon-ng](https://github.com/lanmaster53/recon-ng) - Full-featured web reconnaissance framework.
- [Amass](https://github.com/owasp-amass/amass) - Network mapping and external asset discovery.
- [Subfinder](https://github.com/projectdiscovery/subfinder) - Fast passive subdomain enumeration tool.

## 🔭 Scanning & Enumeration

- [Nmap](https://nmap.org) - The gold standard for network discovery and security auditing.
- [Masscan](https://github.com/robertdavidgraham/masscan) - Fastest internet port scanner.
- [Nikto](https://github.com/sullo/nikto) - Web server scanner that detects dangerous files and misconfigurations.
- [Gobuster](https://github.com/OJ/gobuster) - Directory/file and DNS busting tool written in Go.
- [ffuf](https://github.com/ffuf/ffuf) - Fast web fuzzer written in Go.
- [Enum4linux](https://github.com/CiscoCXSecurity/enum4linux) - Tool for enumerating information from Windows and Samba systems.
- [WhatWeb](https://github.com/urbanadventurer/WhatWeb) - Web application fingerprinting tool.

## 💥 Exploitation Frameworks

- [Metasploit Framework](https://github.com/rapid7/metasploit-framework) - World's most used penetration testing framework.
- [BeEF](https://beefproject.com) - The Browser Exploitation Framework.
- [SQLmap](https://sqlmap.org) - Automated SQL injection and database takeover tool.
- [Impacket](https://github.com/fortra/impacket) - Collection of Python classes for network protocols.
- [Responder](https://github.com/lgandx/Responder) - LLMNR, NBT-NS and MDNS poisoner.
- [CrackMapExec](https://github.com/byt3bl33d3r/CrackMapExec) - Swiss army knife for pentesting Windows/Active Directory environments.

## 🔑 Password Attacks

- [Hashcat](https://hashcat.net/hashcat/) - World's fastest and most advanced password recovery utility.
- [John the Ripper](https://github.com/openwall/john) - Open-source password security auditing tool.
- [Hydra](https://github.com/vanhauser-thc/thc-hydra) - Fast and flexible online password cracking tool.
- [Medusa](https://github.com/jmk-foofus/medusa) - Speedy, massively parallel, modular login brute-forcer.
- [RockYou wordlist](https://github.com/brannondorsey/naive-hashcat/releases/download/data/rockyou.txt) - The classic password wordlist.
- [SecLists](https://github.com/danielmiessler/SecLists) - Collection of multiple types of lists for security assessments.
